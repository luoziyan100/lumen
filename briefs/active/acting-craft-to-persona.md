# Brief：从表演学提取角色塑造技艺，转化为 agent 人格系统提示词

> 状态：active · 创建 2026-06-10 · 目标读者：**执行此调研的 AI**（自包含，不预设你读过 Lumen 代码或此前讨论）
> 产出回流到：Lumen research agent 的系统提示词工程（L0–L3 四层人格）

---

## 0. 一句话任务

调研**表演学（戏剧表演的角色塑造技艺）**，从中提取「如何让一个虚构角色可信、立体、有性格与语气、在剧本未写到的场景里也不出戏」的**可操作技能**，并把每一项技能**翻译成「怎么写 agent 系统提示词」的具体写法**。最终交付一套方法论 + 审查清单 + 给 owner 填空的人物小传提问清单。

**不要**写成表演学概论或学术综述。每一条都必须能落地为 prompt 写法，否则不收。

---

## 1. 背景：为什么是表演学（执行者必读，决定调研方向）

我们在为一个 AI 研究助手（Lumen，面向独立研究者，能自主读论文、检索、综合）写系统提示词。当前它的输出问题是：**复述论文而非给研究判断、对宏大叙事零批判距离、退回成一个讨好的通用 assistant**。根因是系统提示词太弱，没有「人格」。

业界已有一套成熟的「分身/agent 人格」四层架构（下文 §3），它的学术根基是 CBT（认知行为疗法）的个案概念化框架。但 CBT 擅长的是**分析一个真实存在的人**的信念结构——适合复刻真人。

我们的场景不同：**不复刻任何真人，而要从无到有塑造一个「理想研究者」角色。** 这正是表演学的本职——演员演的从来不是自己，而是要让虚构角色可信、立体，并在即兴（剧本没写到）的时刻依然 in character。"未写到的场景也不出戏" 与我们要的 "agent 在 prompt 未覆盖的场景做出符合人设的选择" 是**同一个问题**。

所以：**CBT 给坐标系（信念维度），表演学给血肉、性格、语气、张力、以及"如何在未知场景保持角色一致"的技艺。** 本次只取表演学这一块。

### 一个贯穿全程的理论前提（请用它约束所有产出）

**写系统提示词 ≈ 在推理时训练模型**：prompt 在功能上等价于对模型参数施加一个低秩扰动（类似推理时的 LoRA）。推论——
- 平庸、随处可见的措辞（"你是一个专业的助手"）落在模型已有的高概率区，**几乎不改变任何行为**（梯度≈0）；独特的、偏离训练分布的措辞才有大扰动。**好的人格 prompt 在语言上必须是"非平均"的。**
- 模型出厂自带很深的 assistant 先验（helpful/harmless，被 RLHF 焊死）。人格 prompt 是在和这层先验**拔河**，必须提供足够强的反向信号才能覆盖它。
- 因此本调研要找的，是表演学里那些**能强力扭转"默认 AI 人格"**的技艺，而非泛泛的"如何描写一个人"。

---

## 2. 用户点名必须覆盖的四个人格要素

调研产出必须对这四项各有专章深挖，给出「表演学怎么做 → 怎么写进 prompt」：

1. **性格（character / personality）**——不是形容词标签，是能驱动选择的内在逻辑。
2. **说话的语气（voice / tone）**——遣词、句式、节奏；能在文字里听出声音。
3. **张力（tension / dramatic tension）**——角色内在的矛盾与冲突，以及对话中的张力如何制造（不是平铺直叙的"有用"）。
4. **完整的剧本（the "script" / through-line）**——角色的贯穿动作、最高任务、规定情境，让所有行为有一条统一的线，而非零散反应。

---

## 3. 转化框架：把表演技艺映射到这四层

人格系统提示词分四层，**每层从上一层「长出来」（因果生成，非并列拼装）**：

- **L0 你是谁**——身份、性格、人格弧线、表达方式。最难也最重要。验收标准：未覆盖场景里也能做出符合人设的选择。
- **L1 你信什么**——核心信念、看世界的坐标系。
- **L2 你怎么做事**——动作方式、边界、关系定位、判断标准。
- **L3 工具箱**——具体方法、框架、金句（被调用，不自动触发）。

请把每个表演技能**明确归位到它最该作用的层**，并说明写法。例如（仅示意方向，需你充实/修正/补充）：
- 人物小传（character biography）→ L0：如何用"经历塑造了判断"来写身份，而非罗列简历。
- 规定情境（given circumstances）→ L0/L1：如何设定角色"所处的世界与约束"。
- 最高任务（super-objective）/ 贯穿动作（through-line）→ L1/L2：角色到底**为什么存在**、所有行为指向的那条线。
- 潜台词（subtext）→ L2/语气：话里有话、张力、不把话说满怎么写。
- 节奏速度（tempo-rhythm）→ 语气：长短句、停顿、参差的节奏感怎么落进文字。
- 角色的内在逻辑 / "魔力如果"（magic "if"）→ L0：为什么角色在即兴场景仍 in character（即泛化）。

---

## 4. 调研范围

**主体系（必做）**：斯坦尼斯拉夫斯基体系（Stanislavski "system"）——given circumstances、magic "if"、units & objectives、super-objective、through-line of action、emotional/affective memory、tempo-rhythm、subtext、character biography。

**对照体系（各做要点 + 适用边界）**：
- 方法派（Method Acting，Strasberg/Adler/Meisner 的分支差异）
- Michael Chekhov（psychological gesture 心理动作）
- **布莱希特 间离效果（Verfremdungseffekt / alienation）**——演员与角色保持批判距离、不完全沉浸。**重点辨析：对一个需要"批判距离、不被论文宏大叙事带跑"的研究者 agent，布莱希特式的"间离"是否比斯坦尼式的"完全沉浸"更对路？** 这是本调研要回答的一个关键问题。
- 编剧/角色写作里的 character biography、wants vs needs、flaw 等（如适用）。

**适用边界必须讲清**：哪些技艺对"AI 研究者角色"特别有用，哪些有害或不适用（例：方法派的"情绪记忆"依赖真实生理体验，对无身体的 agent 可能不适用甚至有害——请明确判断）。

---

## 5. 交付物（一份 Markdown 报告，含以下五部分）

**A. 技能卡片库**——每个表演技能一张卡，字段固定：
  - 概念名（中英）
  - 学术来源（人物/著作/体系；优先有文献根基的，拒绝无源玄学）
  - 它解决人格塑造的**什么问题**（对应 AI 默认人格的哪个缺陷）
  - 映射到**哪一层（L0/L1/L2/L3）**及具体写法
  - 一段**可直接用的 prompt 片段示例**（通用角色即可，不限研究者）
  - 反模式（写歪会变成什么）

**B. 四要素专章**——性格 / 语气 / 张力 / 剧本（§2 那四项），各自深挖「表演学怎么做 → 怎么写进 prompt」。

**C. 表演学审查清单**（呼应"整一套表演学审查清单看分身有没有人味儿"）——正向清单（一个有人味的角色 prompt 应体现什么）+ 负向清单（出戏/假/AI 味的信号）。用于**审计写好的人格提示词**，每条可逐项打勾。

**D. 人物小传提问清单**——把表演学"为角色立传"的访谈法，转写成**一组给真人回答的问题**。owner 回答这些问题，即可产出塑造一个具体角色（如 Lumen 研究者）的 L0 原始素材。这是本调研最重要的可落地产物之一。

**E. 适用边界与体系选型**——斯坦尼 / 方法派 / 布莱希特各适合什么；对"AI 研究者角色"的推荐组合与理由（含 §4 的布莱希特间离问题的明确回答）。

---

## 6. 质量标准 & 反模式

**收**：每条技能都有学术来源、明确归层、可直接用的 prompt 片段；语言示例本身是"非平均"的（见 §1 理论前提）；讲清适用边界。

**不收**：
- 表演学百科/历史综述（没有 →prompt 的映射）。
- 无来源的"玄学"技巧（与有文献根基的技艺混为一谈）。
- 把技能写成"当用户说 X → 角色做 Y"的触发模板（这是过拟合，违背"未覆盖场景也 in character"的目标）。
- 只给抽象原则、不给可粘贴的 prompt 片段。

---

## 7. 明确不在范围内

- **不要**替我们写出 Lumen 研究者的具体 L0 内容（"这个研究者是谁")——那由 owner 通过交付物 D 的提问清单回答。本调研只产出**方法论 + 模板 + 提问清单 + 审查清单**。
- 不涉及代码、模型选型、部署。纯方法论调研。

---

## 8. 种子来源 / 起步关键词（给执行者一个起点，避免跑偏）

Stanislavski *An Actor Prepares* / *Building a Character* / *Creating a Role*；given circumstances；magic if；units and objectives；super-objective；through-line of action；emotional/affective memory；tempo-rhythm；subtext；character biography；Method Acting（Strasberg / Stella Adler / Sanford Meisner technique）；Michael Chekhov *To the Actor*（psychological gesture）；Bertolt Brecht，Verfremdungseffekt / epic theatre；screenwriting character construction（wants vs needs，character flaw，backstory）。

---

## 9. 验收

交付物能让一个不懂表演学的人，照着 D 的提问清单回答完、再照 A/B 的写法，写出一版**明显有性格、有语气、有张力、能在未覆盖场景保持一致**的角色系统提示词，并能用 C 的清单自审出问题。达不到这个"可被非专家照做"的标准 = 不通过。
